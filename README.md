# 中医咨询健康网站

## 项目介绍

这是一个基于Flask框架开发的中医咨询健康网站，旨在为用户提供中医问诊、艾灸知识和在线预约服务。网站风格简洁、古朴，符合中医文化的传统特色。

## 功能特点

1. **中医问诊**：用户可以填写详细的症状描述、个人信息（姓名、手机、出生年月日时），提交后由中医师进行诊断。
2. **艾灸知识**：提供艾灸的历史、原理、方法、功效和注意事项等详细信息。
3. **在线预约**：用户可以选择中医师、预约时间，提交预约申请。

## 技术栈

- **后端**：Flask 3.1.3
- **前端**：HTML5、CSS3
- **表单处理**：Flask-WTF 1.2.2、WTForms 3.2.1

## 安装步骤

1. **克隆项目**
   ```bash
   git clone <项目地址>
   cd 中医咨询健康网站
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   # 或
   pip install flask flask-wtf wtforms
   ```

3. **运行项目**
   ```bash
   python app.py
   ```

4. **访问网站**
   打开浏览器，访问 http://127.0.0.1:5000

## 网站使用指南

1. **首页**：了解网站的主要功能和中医文化介绍。
2. **中医问诊**：填写症状描述和个人信息，提交后等待中医师联系。
3. **艾灸知识**：浏览艾灸的相关知识，了解艾灸的功效和使用方法。
4. **在线预约**：选择中医师和预约时间，提交预约申请。

## 注意事项

- 本网站仅作为中医咨询的平台，不能替代专业医生的诊断和治疗。
- 如有紧急健康问题，请立即就医。
